PlayMafia

A Web Based Multiplayer RPG

Created for CS 196[The Foundry: Honors], PlayMafia is a web game that uses HTML, CSS, and Javascript for front end, and a Flask framework in Python for the backend. Based off the game mafia, but with more complexities in gameplay.

Want to try it out? You can visit the site and play yourself at playmafia.me.

  • Date: Fall 2014
  • Client: CS196
  • Category: Web Based Game

STEM Fact Of The Day

A JAVA Web Applet

This JAVA web applet was created for my internship at St. Thomas Aquinas College in Rockland County, NY. For each calander date it will give you a unique STEM Face Of The Day. It also allows you to look through other dates and read up on more facts!

You can try the code for yourself on My Github Page.

  • Date: Spring 2014
  • Client: STAC Internship
  • Category: Web Applet

GnomenWald

A JAVA Simulator

GnomenWald is a economy and transportation simulator. It replicates human desires to gain economic benefit by moving from one place to another. The program does this by utilizing Data Structures, Discrete Mathematics, and several different algorithms.

You can try the program for yourself on My Github Page.

  • Date: Summer 2013
  • Client: CS 2110
  • Category: JAVA Simulator

The Rain Game

A JAVA Game

The Rain Game is a game based in Java. Created for Extra Credit, this program uses basic programming skills to create a simple game to help stroke patients. They are required the type in the numbers on the screen while the game continuously increases in difficulty.

You can try the game yourself on My Github Page.

  • Date: Fall 2014
  • Client: CS 125
  • Category: JAVA Game

EmergEnsure

A Mobile Application Concept

Inspired by Hurricane Sandy, EmergEnsure is a mobile application concept created for the Verizon App Challenge. It utilizes statistical multiplexing to lower mobile traffic during times of need.

You can watch the video explaining the conept here.

  • Date: Winter 2013
  • Client: Verizon
  • Category: Mobile Application Concept
